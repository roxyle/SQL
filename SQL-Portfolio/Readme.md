Progetti SQL:
1. E-commerce Analytics
Analisi dati per azienda e-commerce: identificazione dei clienti più redditizi, dei prodotti invenduti, delle performance delle categorie e dei trend stagionali.
A tal fine ho utilizzato Query SQL con JOIN multipli, aggregazioni e calcoli statistici, subquery e query nidificate e analisi temporale

Cosa troverete all'interno del progetto: DB di 5 tabelle relazionali, 5 query a scopo business

→ Vai al progetto

2. Sistema Logistica/Magazzino
Ottimizzazione gestione inventario, fornitori e supply chain per azienda di distribuzione.
A tal fine ho utilizzato Query SQL con JOIN multipli, CASE WHEN per logica condizionale, calcoli su date (DATEDIFF, MAX su date), LEFT JOIN e gestione valori NULL, HAVING per filtri su aggregazioni

Cosa troverete all'interno del progetto: DB di 8 tabelle relazionali, 4 query per l'identificazione di fornitori affidabili, prodotti critici e movimentazione

Competenze tecniche utilizzate per questi progetti:

Database creato con MAMP (MySQL + phpMyAdmin),
