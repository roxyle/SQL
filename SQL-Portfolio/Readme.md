# Progetti SQL:
### 1. E-commerce Analytics
Analisi dati per azienda e-commerce: identificazione dei clienti più redditizi, dei prodotti invenduti, delle performance delle categorie e dei trend stagionali.
A tal fine ho utilizzato Query SQL con JOIN multipli, aggregazioni e calcoli statistici, subquery e query nidificate e analisi temporale

Cosa troverete all'interno del progetto: readme per descrizioni e istruzioni d'uso, DB con 5 tabelle relazionali, 5 query a scopo business

### 2. Sistema Logistica/Magazzino
Ottimizzazione gestione inventario, fornitori e supply chain per azienda di distribuzione.
A tal fine ho utilizzato Query SQL con JOIN multipli, CASE WHEN per logica condizionale, calcoli su date (DATEDIFF, MAX su date), LEFT JOIN e gestione valori NULL, HAVING per filtri su aggregazioni

Cosa troverete all'interno del progetto: readme per descrizioni e istruzioni d'uso, DB con 8 tabelle relazionali, 4 query per l'identificazione di fornitori affidabili, prodotti critici e movimentazione merci

## Competenze tecniche utilizzate per questi progetti:
Database creato con MAMP (MySQL + phpMyAdmin), queries complesse con JOIN multipli, aggregazioni (SUM, COUNT, AVG, MAX, MIN), subquery e query nidificate, LEFT/INNER JOIN, CASE WHEN per logica condizionale, GROUP BY e HAVING, funzioni su date e calcoli temporali, ottimizzazione per leggibilità.

#### Note:
Tutti i dati utilizzati all'interno delle tabelle dei due progetti (come nomi, indirizzi, ecc..) sono fittizi, generati tramite AI per rappresentare scenari business realistici. 
I progetti li ho realizzati per testare le mie competenze SQL e la mia capacità di analisi dati in contesti professionali.
